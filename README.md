# Kano Event - Venue Marketplace

The "Booking.com for Event Venues" in Northern Nigeria. Built with Next.js 14, TypeScript, Tailwind CSS, and Prisma ORM.

## Features
- Smart Search by LGA and Capacity
- Premium Venue Detail Pages
- Operator Dashboard for Analytics
- Role-based Registration (Customer / Operator)
- Responsive Modern UI

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Prisma ORM + PostgreSQL
- Deployed on Render

## Deployment
Configured for Render via render.yaml blueprint — provisions the web service and PostgreSQL database automatically on push.
